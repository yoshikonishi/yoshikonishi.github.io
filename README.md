# Yoshifumi Konishi Academic Website

This is a lightweight static website for GitHub Pages.

## Pages

- `index.html`: home page
- `cv.html`: curriculum vitae
- `research.html`: research and publications

## Reference Direction

The layout follows common economics and academic GitHub Pages patterns: simple typography, restrained navigation, a compact profile column, and publication-first content. The initial design direction was informed by academic GitHub Pages templates such as Academic Pages and al-folio, while keeping the implementation dependency-free for easy GitHub Pages hosting.
